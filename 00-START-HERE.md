## 🎯 Quick Start - Do This NOW

### Step 1: Import Database (2 minutes)
```
1. Open phpMyAdmin
2. Select 'nexsoft_hub' database
3. Click "SQL" tab
4. Go to: NexSoft/database/design_templates.sql
5. Copy entire contents
6. Paste into SQL tab
7. Click "Execute"
8. ✓ 5 new tables created!
```

### Step 2: Verify Admin Menu (1 minute)
```
1. Go to admin panel
2. Look in left sidebar
3. Under "HR Management" section
4. Should see 3 new items:
   → Design Templates
   → Design Assets
   → Generate Document
5. ✓ Navigation ready!
```

### Step 3: Create First Certificate (10 minutes)
```
1. Click "Design Assets" → "Logos & Images"
2. Upload your company logo (PNG recommended)
3. Click "Design Templates" → "Create New Template"
4. Fill in:
   - Name: "My First Certificate"
   - Type: Certificate
   - Upload logo
5. In Body section, add:
   "[Recipient Name] has completed the course [Course Name]"
6. Click Save
7. Click "Generate Document"
8. Select your template
9. Fill in recipient info
10. Click Generate
11. ✓ Certificate created!
```
