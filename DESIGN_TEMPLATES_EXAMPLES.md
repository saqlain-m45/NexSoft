# Design Templates - Example Templates

Use these examples to quickly set up common certificate and letter templates in NexSoft.

## 📜 Example 1: Course Completion Certificate

### Basic Information
- **Template Name:** Course Completion Certificate
- **Type:** Certificate
- **Description:** Standard certificate for course completion

### Header HTML
```html
<div style="text-align: center; margin-bottom: 30px;">
    <p style="font-size: 10px; letter-spacing: 2px; color: #666;">THIS CERTIFICATE IS PROUDLY PRESENTED TO</p>
</div>
```

### Body HTML
```html
<div style="text-align: center; margin: 40px 0;">
    <h1 style="font-family: Georgia, serif; font-size: 32px; color: #1a1a1a; margin: 20px 0;">
        <strong>[Recipient Name]</strong>
    </h1>
    
    <p style="font-size: 16px; color: #333; margin: 30px 0;">
        For successfully completing the course
    </p>
    
    <h2 style="font-family: Georgia, serif; font-size: 24px; color: #0066cc; margin: 20px 0;">
        [Course Name]
    </h2>
    
    <p style="font-size: 14px; color: #555; margin: 20px 0; line-height: 1.8;">
        This certifies that the above-named candidate has demonstrated exceptional proficiency
        in the course and is hereby awarded this certificate of completion.
    </p>
    
    <table style="width: 100%; margin-top: 50px; font-size: 14px;">
        <tr>
            <td style="width: 50%; text-align: center;">
                <p style="margin: 20px 0 5px 0;">Certificate ID: <strong>[Certificate ID]</strong></p>
                <p style="margin: 5px 0 0 0;">Issued: [Issue Date]</p>
            </td>
            <td style="width: 50%; text-align: center;">
                <p style="margin: 20px 0 5px 0;">Duration: [Duration]</p>
                <p style="margin: 5px 0 0 0;">Grade: [Grade]</p>
            </td>
```

## 📄 Example 2: Internship Experience Letter

### Basic Information
- **Template Name:** Internship Experience Letter
- **Type:** Letter
- **Description:** Professional experience letter for interns

```html
<div style="margin-bottom: 30px;">

    Date: [Issue Date]
</p>

<p style="margin: 20px 0; color: #333;">
```

```html

    and contributed significantly to the project's success.
</p>

<p style="font-size: 14px; line-height: 1.8; text-align: justify; margin: 20px 0; color: #333;">
    Key achievements and skills demonstrated:
</p>

<ul style="font-size: 14px; line-height: 1.8; margin: 20px 0; color: #333; margin-left: 30px;">
    <li>Proficiency in [Technical Skills]</li>
    <li>Strong problem-solving abilities</li>
    <li>Excellent team collaboration</li>
    <li>Professional communication skills</li>

<p style="font-size: 14px; line-height: 1.8; text-align: justify; margin: 20px 0; color: #333;">
    We confirm that during the internship, [Recipient Name] was a valuable team member and displayed 
    commendable performance. We wish [him/her] the very best in his/her future endeavors.
</p>

<p style="font-size: 14px; line-height: 1.8; margin: 20px 0; color: #333;">

</p>
```

## 📋 Template Variables Quick Reference

Copy any of these examples and replace the variable placeholders:

| Variable | Usage | Example |
|----------|-------|---------|
| `[Recipient Name]` | Person receiving certificate | John Doe |
| `[Certificate ID]` | Auto-generated unique ID | A1B2C3D4 |
| `[Issue Date]` | Date certificate issued | 2024-03-25 |
| `[Course Name]` | Name of course/program | React.js Fundamentals |
| `[Duration]` | Length of program | 4 Weeks |
| `[Grade]` | Final grade/performance | A+ |
| `[Instructor Name]` | Name of instructor | Dr. Sarah Khan |
| `[Department]` | Department/Division | Web Development |
| `[Project Name]` | Project worked on | E-Commerce Platform |
| `[Achievement Description]` | What achievement is for | Outstanding Performance |
| `[Credential Title]` | Professional credential | Certified Web Developer |
| `[Skills/Competencies]` | Technical skills | HTML, CSS, JavaScript, React |
| `[Training Name]` | Name of training | Advanced JavaScript |

## 🎨 Styling Tips

### Text Sizes
```
Small: font-size: 12px
Normal: font-size: 14px
Medium: font-size: 16px
Large: font-size: 20px
Extra Large: font-size: 24px+
```

### Colors
```
Professional Blue: #0066cc
Dark Text: #333333 or #1a1a1a
Light Gray: #666666 or #999999
Accent: #0066cc or #28a745 (green)
```

### Spacing
```
Vertical space: margin: 20px 0 (or 30px, 40px)
Horizontal padding: padding: 0 20px
Line height: line-height: 1.6 (comfortable reading)
Text alignment: center, left, right, justify
```

### Fonts
```
Serif (professional): Georgia, serif
Sans-serif (modern): Arial, sans-serif
Monospace (tech): 'Courier New', monospace
```

## 💡 Pro Tips

1. **Reusable Variables** - Use the same variables across similar templates
2. **Consistent Styling** - Match colors and fonts to your brand
3. **Logo Placement** - Center-top is most professional
4. **Spacing** - Use generous margins (40-60px) for important elements
5. **Font Sizes** - Main text 14-16px, headings 20-28px, footer 11-12px
6. **Borders** - Add subtle borders (1-2px) for elegance
7. **Tables** - Use for signature lines and multi-column layouts

## 🚀 Getting Started

1. Choose an example above
2. Copy the Header, Body, and Footer HTML
3. Go to Admin → Design Templates → Create New
4. Paste the HTML into respective sections
5. Customize text/colors to match your brand
6. Upload your logo
7. Save and use!

Each example is designed to work immediately and can be customized further with different variables and styling.
